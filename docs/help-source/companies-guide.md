# 분석 문서: /companies — 회사 목록

작성일: 2026-06-17 | 분석 기반: 코드 (git main)

---

## 1. 페이지 위치·역할

- **URL**: `/companies`
- **사이드바 메뉴**: 회사 목록 (IcCompany 아이콘)
- **권한**: 로그인 사용자 전체
- **기본값**: 매출액 내림차순 / 전체 규모 / 상장+비상장

무신사에 입점한 브랜드를 보유한 법인(회사) 목록을 재무·랭킹 데이터와 함께 제공하는 페이지.
DART 공시 재무정보와 무신사 랭킹 데이터를 결합해 경쟁사 분석에 활용.

---

## 2. 화면 구성 (코드 기반)

### 좌측: 필터 레일
- **검색**: 회사명 검색 (350ms 디바운스)
- **정렬**: 매출액 / 랭킹상품수 / 이름순 (CompanySortKey)
- **상장 구분**: 상장 / 비상장 (복수 선택)
- **기업 규모**: 대기업(1천억+) / 중견기업(100억~1천억) / 소기업(100억 미만) / 미수집(DART 없음)
- **옵션**: 재무정보 있는 회사만 / 자사 브랜드 보유 회사만

### 우측: 회사 테이블
- 컬럼: 로고 / 회사명 / 랭킹 상품수 / 매출액·성장률 / 영업이익률 / 부채비율 / 상장구분
- 100개씩 페이지네이션
- 행 클릭 → `/company?id=법인ID` 회사 상세

### 우상단: CSV 다운로드

---

## 3. 데이터 흐름

- `fetchCompanyPage()`: 서버 사이드 필터·정렬 (DB 쿼리)
- 페이지 이탈 후 복귀해도 필터 유지: sessionStorage 저장 (SS_KEY='companies-state-v2')
- 검색어 변경 시 page 자동 리셋

---

## 4. 사용자 인터랙션

- **검색창 입력** → 350ms 후 자동 검색
- **행 클릭** → `/company?id=XXX` 회사 상세
- **필터 변경** → 즉시 결과 업데이트
- **[초기화]** → 모든 필터 기본값 복원

---

## 5. 빈 상태·에러 처리

- 검색 결과 없음: 빈 테이블
- 데이터 로드 실패: 에러 메시지 표시

---

## 6. 관련 기능

- 회사 행 클릭 → `/company?id=XXX` (DART 재무 상세, 브랜드 목록)
- 상품 랭킹 → 회사 필터로 특정 회사 브랜드만 조회 가능

---

## 7. 사용자가 자주 헷갈리는 부분

- "매출액이 없는 회사가 많은 이유?" → DART(금융감독원 전자공시시스템)에 공시 의무가 없는 소규모 법인은 재무 데이터가 없음
- "'미수집' 규모 선택 시?" → DART 공시가 없는 회사만 표시됨
- "자사 브랜드 보유 회사란?" → B.CAVE가 직접 운영하는 브랜드(커버낫, 리 등)를 보유한 법인
- "브랜드와 회사의 차이?" → 브랜드=상품 레이블(예: 커버낫), 회사=법인(예: 주식회사 에이비씨)

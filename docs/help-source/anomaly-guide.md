# 분석 문서: /anomaly — 이상탐지

작성일: 2026-06-17 | 분석 기반: 코드 (git main)

---

## 1. 페이지 위치·역할

- **URL**: `/anomaly`
- **사이드바 메뉴**: 이상탐지 (IcFlag 아이콘)
- **권한**: 로그인 사용자 전체
- **기본값**: 오늘 / 전 영역 / 전 심각도

시스템이 자동으로 감지한 비정상적인 변화(이상 신호)를 모아 보는 페이지.
상품 순위 급등·급락, 가격 변동, 리뷰 이상, 프로모션 고할인 등 9종류의 이벤트를 실시간성으로 제공.

---

## 2. 화면 구성 (코드 기반)

### 좌측: 필터 레일
- **기간**: 오늘 / 7일 / 30일 / 90일 / 직접 입력
- **영역**: 상품 / 프로모션 / 리뷰 (복수 선택)
- **심각도**: HIGH / MEDIUM / LOW (복수 선택)
- **저장된 필터**: SavedFiltersDropdown

### 우측: 이상탐지 목록
- 각 행: 심각도 뱃지 / 영역 태그 / 이상 유형 / 이벤트 설명 / 엔티티(상품/브랜드명) / 감지 시각
- 행 클릭 → 우측 드로어 열림

### 상세 드로어 (행 클릭 시)
- 심각도 / 영역 / 감지 시각
- 이상 유형 + 상세 설명
- 연관 엔티티 링크 (상품 → `/product`, 브랜드 → `/brand`)
- **핵심 지표**: 이상 유형에 맞는 수치 (이전→현재 변화)
- **이벤트 히스토리**: 타임라인
- **메모**: 해당 이상탐지에 메모 첨부
- 이전/다음 탐색 버튼

---

## 3. 데이터 흐름

- 이상탐지 이벤트 유형 9가지:
  - `rank_spike` (순위 급등)
  - `rank_drop_own` (자사 순위 이탈)
  - `new_entrant_top10` (TOP10 신규 진입)
  - `sold_out` (품절 전환)
  - `price_drop` (가격 급락)
  - `promo_heavy_discount` (고할인 프로모션)
  - `review_count_surge` (리뷰 폭증)
  - `review_rating_drop` (별점 급락)
  - `review_negative_surge` (부정 리뷰 급증)
- 영역별 분류: 상품(rank_spike/new_entrant_top10/rank_drop_own/sold_out/price_drop), 프로모션(promo_heavy_discount), 리뷰(나머지)
- 심각도: HIGH(빨강) / MEDIUM(노랑) / LOW(회색)
- 실시간 갱신 없음 (필터 변경 시 재쿼리)

---

## 4. 사용자 인터랙션

- **행 클릭** → 상세 드로어 열기
- **드로어 내 엔티티 링크** → 해당 상품/브랜드 페이지
- **이전/다음 버튼** → 같은 목록 내 탐색
- **메모 버튼** → 해당 이상탐지에 메모 추가
- **[초기화]** → 필터 리셋

---

## 5. 빈 상태·에러 처리

- 선택 기간에 이상탐지 없음: 빈 목록 ("no results" 태그)

---

## 6. 관련 기능

- 상품 이상탐지 → `/product?no=XXX` 바로 이동 가능
- 브랜드 이상탐지 → `/brand?id=XXX`
- 홈 대시보드 → 이상탐지 최근 신호 위젯
- 오늘의 매거진 → 이상 탐지 카드

---

## 7. 사용자가 자주 헷갈리는 부분

- "HIGH/MEDIUM/LOW 기준?" → 변화 폭의 크기 기반 자동 분류. HIGH는 즉각 대응이 필요한 수준
- "어제 이상탐지가 오늘도 보여?" → 기간 필터로 조회 범위 조절 가능. '오늘' = 당일 감지된 것만
- "같은 상품이 반복 감지될 수 있나요?" → 같은 날 다른 이상 유형으로 복수 탐지 가능
- "드로어에서 상품 링크가 없는 경우?" → 엔티티 정보가 아직 수집되지 않은 경우 링크가 표시되지 않음

# 분석 문서: /reviews — 자사 리뷰

작성일: 2026-06-17 | 분석 기반: 코드 (git main)

---

## 1. 페이지 위치·역할

- **URL**: `/reviews`
- **사이드바 메뉴**: 자사 리뷰 (IcReview 아이콘)
- **권한**: 로그인 사용자 전체
- **기본값**: 최근 30일 / 전체 상품 / 전체 카테고리

B.CAVE 자사 브랜드 상품에 달린 무신사 리뷰를 모아 볼 수 있는 페이지.
별점, 리뷰 본문, 구매자 속성(사이즈·퀄리티 등), 사진을 함께 확인 가능.
CS 이상탐지(별점 급락, 부정 리뷰 급증 등)도 우측 사이드바에서 함께 표시.

---

## 2. 화면 구성 (코드 기반)

### 좌측 상단: 기간·필터 영역
- **기간**: 오늘 / 7일 / 30일 / 90일 / 직접 입력
- **브랜드**: 자사 브랜드 선택 (단일)
- **상품**: 브랜드별 상품 선택 (단일)
- **카테고리**: 전체 카테고리 복수 선택
- **별점**: 1~5점 각각 선택
- **속성 필터**: 사이즈·퀄리티·신축성·두께감·보온성·무게감·착용감·색감 (각 항목별 답변 선택)
- **키 높이·몸무게**: 슬라이더 범위 필터 (150~195cm, 45~110kg)
- **사진 있는 리뷰만** 체크박스

### 중앙: 리뷰 카드 목록
- 각 카드: 상품 썸네일 / 별점(1~2=빨강, 3=노랑, 4~5=초록) / 날짜 / 도움됨 수 / 사진 수
- 리뷰 본문
- 속성 응답 (사이즈/퀄리티 등)
- 구매자 키·몸무게 정보
- 사진 클릭 → 라이트박스
- 메모 버튼: 해당 리뷰에 메모 첨부

### 우측: CS 이상탐지 패널
- CS 관련 이상탐지(별점 급락·부정 리뷰 급증 등)를 실시간으로 표시
- 각 이상 항목: 이벤트 이름 / 심각도 / 감지 시각

### 우상단: 엑셀 내보내기

---

## 3. 데이터 흐름

- `fetchReviews()`: 선택 필터로 리뷰 조회
- `fetchReviewStats()`: 전체 통계 (총 건수, 평균 별점, 저점 비율)
- `fetchCsAnomalies()`: CS 이상탐지 신호
- `fetchOwnProducts()`: 자사 상품 목록 (브랜드 선택용)
- localStorage 필터 자동 저장

---

## 4. 사용자 인터랙션

- **브랜드/상품 선택** → 해당 상품 리뷰만 표시
- **별점 필터** → 원하는 점수 리뷰만 조회
- **속성 필터** → 사이즈·퀄리티 등 구매 후기 항목별 조회
- **사진 클릭** → 라이트박스 확대
- **메모 버튼** → 해당 리뷰에 내부 메모 추가
- **엑셀 내보내기** → 현재 필터 결과를 엑셀 파일로 저장

## 6. 관련 기능

- CS 이상탐지 → `/anomaly` 에서도 확인 가능
- 오늘의 매거진(CS탭) → 오늘의 리뷰/저점 패턴/고점 패턴/문제 상품 카드

---

## 7. 사용자가 자주 헷갈리는 부분

- "속성 필터란?" → 무신사 리뷰에서 구매자가 답하는 사이즈·퀄리티 등 만족도 항목
- "사진 있는 리뷰만 보는 방법?" → 필터에서 '사진 리뷰만' 체크박스 활성화
- "CS 이상탐지 패널이 안 보여?" → 최근 이상탐지가 없는 경우 패널이 비어있음
- "키·몸무게 필터가 적용 안 되는 경우?" → 구매자가 해당 정보를 기입한 리뷰에만 적용됨

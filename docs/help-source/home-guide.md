# 분석 문서: / — 홈 대시보드

작성일: 2026-06-17 | 분석 기반: 코드 (git main)

---

## 1. 페이지 위치·역할

- **URL**: `/`
- **사이드바 메뉴**: 홈 (IcHome 아이콘)
- **권한**: 로그인 사용자 전체
- **기본값**: 없음 (모든 데이터 자동 로드)

로그인 후 가장 먼저 표시되는 대시보드. 수집 현황 KPI, 오늘의 상품랭킹 TOP10, 브랜드랭킹 TOP10, 자사 브랜드 현황, 이상탐지 신호, 리뷰 통계, 프로모션 요약을 한 화면에서 제공.

---

## 2. 화면 구성 (코드 기반)

### KPI 스트립 (6개 카드)
- 상품 랭킹 스냅샷 수
- 브랜드 랭킹 건수
- 자사 SKU 수 + 브랜드 수
- TOP100 진입 수
- 수집 리뷰 수 + 평균 별점
- 프로모션 수 + 모듈 수
- 각 카드 클릭 → 해당 페이지 이동

### 상품랭킹 TOP10 위젯
- 성별 필터(전체/남성/여성) 전환 가능
- 순위·변동·상품명·할인율·가격 표시
- 자사 상품 'lo' 뱃지 표시
- 행 클릭 → `/product?no=XXX`
- [전체] 링크 → `/ranking`

### 브랜드랭킹 TOP10 위젯
- 성별 필터 전환 가능
- 브랜드명·TOP100 수·평균 순위 표시
- [전체] 링크 → `/brand-ranking`

### 자사 브랜드 현황 위젯
- 커버낫·리·와키윌리 등 자사 브랜드별 TOP100 진입 수·SKU 수
- 브랜드별 그룹 구분선

### 이상탐지 신호 위젯
- 최근 이상탐지 목록
- 심각도 뱃지 + 이벤트 설명
- [전체] → `/anomaly`

### 리뷰 통계 위젯
- 최근 30일 수집 리뷰 수, 평균 별점, 저점(1~2점) 수
- [전체] → `/reviews`

### 프로모션 위젯
- 현재 진행 중인 프로모션 목록
- 종류: 선착순특가 / 하루특가 / 브랜드위크 / 기획전

### 수집 상태 패널
- 수집 스크립트별 상태(활성/미완/대기) + 최근 수집일 + 건수
- 수집 진행 중일 때 실시간 진행률(%) 표시
- Supabase Realtime 구독 + 5~30초 폴링 병행

---

## 3. 데이터 흐름

- 여러 쿼리 병렬 실행: 수집 통계, 자사 브랜드 현황, 이상탐지, 리뷰 통계, 프로모션
- 랭킹·브랜드랭킹: 성별 변경 시 별도 재쿼리
- 수집 작업: Realtime 구독 + 폴링 (진행 중 5초, 없으면 30초)

---

## 4. 사용자 인터랙션

- **KPI 카드 클릭** → 해당 상세 페이지 이동
- **랭킹 성별 전환** → 즉시 재쿼리
- **상품 행 클릭** → `/product?no=XXX`
- **이상탐지 항목** → `/anomaly` (개별 이동 링크 없음 — [확인 필요])

---

## 5. 빈 상태·에러 처리

- 각 위젯 독립적으로 로딩 처리
- 수집 데이터 없으면 '—' 또는 0 표시

---

## 6. 관련 기능

- 오늘의 매거진 → AI 브리핑 전체 보기
- 상품 랭킹, 브랜드 랭킹, 이상탐지, 리뷰, 프로모션 각 페이지로 바로가기

---

## 7. 사용자가 자주 헷갈리는 부분

- "KPI 숫자가 언제 업데이트되나요?" → 수집 작업 완료 직후 Realtime으로 갱신되거나, 최대 30초 폴링 주기로 갱신됨
- "수집 상태 '미완' 의미?" → 당일 수집이 시작됐지만 아직 완료되지 않은 상태
- "자사 브랜드 현황에 내 브랜드가 없어요?" → 자사 브랜드로 등록된 브랜드만 표시됨 (관리자 설정 필요)
- "모바일에서 홈이 달라 보이는 경우?" → 모바일은 MobileHomeView 컴포넌트로 레이아웃이 다르게 표시됨

# Skill 00 — 스킬 파일 사용 가이드

> Claude Code 가 가장 먼저 읽어야 할 파일.

---

## 스킬 파일이란?

UTTU 프로젝트의 시행착오에서 축적된 **best practice 모음**이다.
이전 프로젝트(MDA)에서 발생한 버그·실수·비효율을 반복하지 않기 위해 만들어졌다.

**스킬 파일 없이 작업 = 같은 실수 반복 보장**

---

## 사용 규칙

1. **작업 시작 전** 관련 스킬 파일 전체 읽기
2. **모르는 것** 있으면 스킬 파일에서 먼저 찾기
3. **스킬 파일과 다른 방법**을 쓸 때는 그 이유를 명시
4. **새로운 패턴 발견** 시 스킬 파일에 추가 (학습 축적)

## 핵심 교훈 요약 (MDA → UTTU)

```
랭킹:   실시간(now) → DAILY period (1회/일)
차원:   없음 → 성별×나이대×카테고리 189조합
리뷰:   집계만 → 텍스트 전체 + 증분
DB:     error 무시 → 반드시 처리
링크:   하드코딩 → routes.ts 상수
CSS:    hex → CSS 변수
기반:   없음 → docs/skills 먼저
```
